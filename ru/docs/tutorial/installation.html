Установка

Вы можете установить WCS в ваш проект тремя разными методами:

С помощью NPM​

Предназначена только для пользователей Тайпскрипта: Установи WCS прописав данную команду в консоль:

npm i @rbxts/wcs@latest

С помощью Wally​

Установи WCS с помощью популярного пакет-менеджера Wally. Помести данную линию под графу [dependencies] в файле wally.toml. Желательно использовать последнюю версию:

wally.toml
[dependencies]
wcs = "cheetiedotpy/wcs@1.0.2"
