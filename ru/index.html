WCS

Гибкий фреймворк для создания Комбат Систем.

Начать

Легок в использовании

WCS - легок в использовании, фреймворк покрывает кучу аспектов хорошей комбат системы и предоставляет абстракции для них.

Гибкий и абстрактный

WCS очень гибкий и абстрактный фреймворк, из-за чего позволяет сделать абсоютно любою Комбат Систему и никак не ограничивает пользователя.

Сделан с помощью TypeScript

WCS разработан с помощью TypeScript и поддерживает его нативно, но он также экспортирует Luau типы для людей, которые не используют TS.
